## Fake News Detection using NLP and Transformers

This project focuses on detecting whether a news article is fake or real using natural language processing. The aim was to compare traditional text classification models with a transformer-based model.

## What This Project Includes
Text cleaning and preprocessing <br />
TF-IDF vectorisation<br />
Training classical machine learning models<br />
Fine-tuning a transformer model<br />
Model evaluation using classification metrics<br />
Explainability using SHAP<br />
Topic modelling<br />
Models Used<br />
Logistic Regression<br />
Naive Bayes<br />
Linear SVM<br />
DistilBERT<br />
K-Means Clustering<br />
Latent Dirichlet Allocation<br />

## What I Learned

This project helped me understand how machine learning and deep learning can be used for text classification. I learned how TF-IDF works, how traditional models perform on text data, and how transformer models such as DistilBERT can improve performance in NLP tasks.
