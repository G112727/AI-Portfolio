## Adult Income Prediction using Machine Learning

This project focuses on predicting whether a person earns more than 50K per year using the Adult Income dataset. The aim of this project was to work with tabular data, clean and prepare the dataset, train different machine learning models, and compare their performance.

## What This Project Includes
Data cleaning and preprocessing <br />
Exploratory data analysis<br />
Feature encoding and scaling <br />
Training multiple machine learning models <br />
Model evaluation using accuracy, precision, recall, F1-score, and ROC-AUC <br />
Artificial Neural Network implementation <br />
Feature importance analysis<br />
Basic unsupervised clustering<br />
