## Urban Sound Classification using Machine Learning and Deep Learning

This project focuses on classifying environmental sounds from audio clips. The aim was to extract useful audio features, train machine learning models, and compare them with a CNN trained on spectrogram-based audio representations.

## What This Project Includes
Audio loading and preprocessing
MFCC feature extraction
Spectrogram generation
Training classical machine learning models
Training a CNN model for audio classification
Model evaluation and comparison
Feature importance analysis
Models Used
Support Vector Machine
Random Forest
XGBoost
Convolutional Neural Network
K-Means Clustering

What I Learned

## This project helped me understand how audio data can be converted into useful features for machine learning. I learned how MFCCs and spectrograms can represent sound patterns, and how CNNs can be used for audio classification tasks.
