## Plant Disease Classification using Deep Learning

This project focuses on classifying plant leaf diseases from image data. The aim was to compare traditional machine learning methods with a deep learning approach using a convolutional neural network.

## What This Project Includes
Image loading and preprocessing
Feature extraction from images
Training traditional machine learning models
Fine-tuning a deep learning model
Model evaluation and comparison
Visual explanation using Grad-CAM
Models Used
Support Vector Machine
Random Forest
k-Nearest Neighbours
ResNet-based Convolutional Neural Network
Tools and Libraries
Python
NumPy
Matplotlib
Scikit-learn
PyTorch
Torchvision
## What I Learned

This project helped me understand how image classification works and why deep learning models are powerful for visual data. I also learned how pretrained CNN models can improve performance compared to manually extracted image features.
